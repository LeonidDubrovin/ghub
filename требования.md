# GHub - Пользовательские требования

## Референсные проекты

- https://github.com/itchio/itch - Официальный клиент itch.io
- https://github.com/JosefNemec/Playnite - Универсальный лаунчер для Windows

---

## 1. Основные функции

### 1.1 Управление библиотекой игр

| ID | Требование | Приоритет |
|----|------------|-----------|
| F1 | Находить игры и помещать их в каталог | Высокий |
| F2 | Получать информацию об игре из каталога с игрой | Высокий |
| F3 | Получать информацию об игре из интернета (IGDB, SteamGridDB) | Высокий |
| F4 | Массовое добавление игр из папки | Высокий |
| F5 | Проверка дубликатов при добавлении | Высокий |
| F6 | Ручное связывание локальной игры с источником метаданных | Средний |

### 1.2 Пространства (Spaces)

| ID | Требование | Приоритет |
|----|------------|-----------|
| S1 | Поддержка игр на разных жёстких дисках | Высокий |
| S2 | Разделение: Steam / itch.io / другие источники | Высокий |
| S3 | При массовом добавлении указывать целевое пространство | Высокий |
| S4 | Визуальное отображение пространств в сайдбаре | Высокий |

**Сценарий использования:**
- D:\Games - основная библиотека
- E:\Indies - инди-игры
- C:\Steam\steamapps - игры Steam
- D:\itch - игры из itch.io
- Ещё 2 каталога с играми из других источников

### 1.3 Интеграция с itch.io

| ID | Требование | Приоритет |
|----|------------|-----------|
| I1 | Авторизация в itch.io (OAuth) | Высокий |
| I2 | Скачивание игр напрямую в выбранный каталог | Высокий |
| I3 | Использование Butler для загрузок | Высокий |
| I4 | Показ прогресса загрузки | Высокий |

### 1.4 Отложенные ссылки (Wishlist)

| ID | Требование | Приоритет |
|----|------------|-----------|
| W1 | Добавление списка ссылок для изучения | Средний |
| W2 | Ссылки идут в отдельную колонку/раздел | Средний |
| W3 | Просмотр и изучение ссылок позже | Средний |
| W4 | Скачивание игры с itch.io из списка в выбранное пространство | Средний |

### 1.5 Учёт времени игры

| ID | Требование | Приоритет |
|----|------------|-----------|
| T1 | Отслеживание времени запуска игры | Высокий |
| T2 | Обновление времени во время игры (heartbeat) | Высокий |
| T3 | Сохранение времени при внезапной остановке системы | Высокий |
| T4 | Максимальная потеря данных: 15-30 секунд | Высокий |

### 1.6 Сопоставление с интернет-источниками

| ID | Требование | Приоритет |
|----|------------|-----------|
| M1 | При добавлении собирать информацию из папки игры | Высокий |
| M2 | Предложить выбрать ресурс для поиска (IGDB, Steam, itch) | Высокий |
| M3 | Найти потенциально подходящие игры | Высокий |
| M4 | Показать пользователю варианты для выбора | Высокий |
| M5 | При подтверждении обновить данные в приложении | Высокий |
