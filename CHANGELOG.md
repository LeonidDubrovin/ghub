# GHub - История изменений

## Реализованные функции

### Phase 1: Foundation ✅
- [x] **Tauri 2.0 + React 18 + TypeScript** — основа приложения
- [x] **SQLite база данных** — схема с таблицами games, spaces, installs, sessions
- [x] **Базовый UI** — сайдбар с пространствами, сетка игр, карточки
- [x] **CRUD операции** — создание, чтение, обновление, удаление игр и пространств
- [x] **Сканирование папок** — рекурсивный поиск игр с обнаружением exe-файлов
- [x] **Запуск игр** — с защитой от повторного запуска
- [x] **Локализация** — русский и английский языки (i18next)

### Phase 1.5: UI Improvements ✅
- [x] **Details View** — новый режим отображения в стиле Playnite
- [x] **Контекстное меню** (ПКМ)
- [x] **Редактирование игр**
- [x] **Удаление** игр и пространств с подтверждением
- [x] **Статусы прохождения**: Не играл, Играю, Пройдено, Брошено, Отложено

- [x] **Portable build** — скрипт build-portable.ps1 для сборки x64

## Технологический стек

| Компонент | Технология |
|-----------|------------|
| Framework | Tauri 2.0 |
| Backend | Rust |
| Frontend | React 18 + TypeScript |
| Bundler | Vite 6 |
| State | Zustand + TanStack Query |
| Styles | Tailwind CSS |
| Database | SQLite (rusqlite) |
| i18n | i18next |
| Scraping | reqwest + scraper |
